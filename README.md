## Installation

1. Clone repo
2. Run `npm install`
3. Create a file in the config/ directory called "database_url.js".
4. Inside that file, write `module.exports = ` followed by your mongo database url, as a string.
## Usage

1. run `node server.js`
2. Navigate to `localhost:8080`
